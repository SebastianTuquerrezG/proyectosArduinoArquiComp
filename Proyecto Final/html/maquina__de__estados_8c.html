◆ MenuConfig()

void MenuConfig ( )

MenuConfig Funcin encargada de mostrar el men de configuracin en la pantalla.

Descripcion

se utiliza la librera "Keypad" para leer la entrada del teclado y realizar acciones en el men. Se pueden navegar entre las diferentes pantallas definidas con la libreria "LiquidMenu", utilizando las teclas "A" para ir a la pantalla anterior y "D" para ir a la siguiente. Al presionar "#" se alterna el enfoque entre los elementos del men y al presionar "*" se activa la funcin asociada al elemento enfocado. Si se cambia al estado de monitoreo de temperatura, la funcin sale y se devuelve el control al sistema.

Note
La funcin se quedar en un bucle infinito hasta que se realice una accin que requiera salir de este estado.

void Seguridad ( )

seguridad Funcin de seguridad para proteger el sistema.

Descripcion

Esta funcin permite al usuario ingresar una clave de seguridad y verifica si es la clave correcta. Si la clave es incorrecta, se muestra el nmero de intentos restantes y se incrementa el contador de intentos. Si se alcanzan 4 intentos fallidos, el sistema se bloquear y se activar una alarma. La funcin tambin se encarga de reiniciar el sistema en caso de bloqueo.
